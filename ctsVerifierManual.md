# Run the test
1. Do the following
   - Enable airplane mode
   - Disable Adaptive Brightness
   - Disable Auto-Rotate
   - Disable Stay awake
   - Disable Location
2. Open ctsVerifierSample and click on the test
3. Follow throught the instrctions.
